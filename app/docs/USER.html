Usage Overview

  1. Choose Campus.
  2. Choose Building.
  3. Choose Floor (if no floors are available or floors are missing then the SVG map needs added to the application).
  4. Viewing the floorplan view will display a visualization of data.
  5. Viewing the table view will display a detail list for the current building.
  6. Changing the date and time values will modify the results to reflect the manual parameters. Clicking on the date triggers a reload.
  7. Notifications will display useful information about what the application is receiving from the user input.
  8. Loading messages will notify the user if data is being processed.
